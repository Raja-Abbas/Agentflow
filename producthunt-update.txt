🚀 AgentFlow Update: Build AI Chatbots Visually — Zero Code, Zero Server Costs

I've been shipping hard on AgentFlow — a completely free, no-code visual chatbot builder. Here's what's new:

🧩 **Visual Flow Editor** — Custom SVG canvas with drag-and-drop nodes (triggers, AI responses, conditions, API calls). No React Flow dependency.

↩️ **Undo/Redo** — Ctrl+Z/Y for every action. Drag, drop, delete, edit — all tracked.

👥 **Collaborative Editing** — Real-time sync across browser tabs via BroadcastChannel. Work with your team instantly.

🎨 **Resizable Panels** — Drag handles between palette, canvas, and properties panels.

🔍 **Live Debug Mode** — Step through nodes, trace decisions, pick condition branches, inspect AI prompts. Color-coded messages for every step.

🎤 **Voice Input** — Test flows hands-free with Web Speech API. Works in conversations and debug mode.

🤖 **Multi-Model LLM** — BYO API key for OpenAI GPT-4, Anthropic Claude, and Google Gemini. Calls go browser→provider directly — zero cost to me, zero cost to you.

📦 **Embeddable Widget** — One `<script>` tag on any website. Floating chat button with popup. Full dark mode support.

📋 **Flow Versioning** — Save named snapshots, browse history, restore any version.

🔗 **Share via URL** — Encode your entire flow in a shareable link.

📊 **Node Usage Analytics** — See which node types you build with most.

📱 **Dark Mode** — Full support across every page. Toggle in the sidebar.

All free, open source, zero server costs. Try it: https://agentflow.rajaabbas.com
